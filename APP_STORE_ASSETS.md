## üì∏ How to Capture Screenshots

### Option 1: Use Your Phone
1. Build and install the app on your device
2. Navigate to each screen
3. Take screenshots using device buttons
4. Transfer to computer
5. Resize if needed using online tools

### Option 2: Use Android Emulator
```bash
# In Android Studio
# Run app in emulator
# Use emulator screenshot tool
# Save to required dimensions
```

### Option 3: Use iOS Simulator
```bash
# In Xcode
# Run app in simulator
# Cmd + S to take screenshot
# Screenshots auto-saved to Desktop
```

### Option 4: Use Expo Go (Quick Testing)
1. Run `expo start` in mobile directory
2. Scan QR code with Expo Go app
3. Navigate through app
4. Take screenshots on your device
